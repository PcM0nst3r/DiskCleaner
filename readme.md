# Disk cleaner

This is a simple script for overwriting the bytes against digital forensics.

***!!! IMPORTANT NOTE: ONLY USE IT ON EXTERNAL DISK DRIVE. DO NOT USE IT ON PRIMARY DISK DRIVE.***

#### Usage 

```
python disk_cleaner.py
```
